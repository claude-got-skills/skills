# Practical Pipeline Architecture

Here's a straightforward approach that scales reasonably without over-engineering:

## Core Components

```
PDF Input → Chunking → Extraction → Cross-reference → Summary
```

**1. PDF Processing**
- Use `pypdf` or `pdfplumber` for text extraction (faster than heavy ML approaches initially)
- Split into logical chunks (pages or sections) rather than arbitrary token counts
- Keep metadata (page numbers, section headers) for traceability

**2. Information Extraction**
- Start with **rule-based extraction** for highly structured data (dates, amounts, names)
- Add LLM extraction for unstructured content (using an API like Claude or open-source models)
- Cache prompts if using an API—same extraction logic across 100 pages saves costs

**3. Knowledge Base Integration**
- SQLite or lightweight PostgreSQL for a small team
- Simple semantic search: use embeddings (openai-embeddings or open-source) + vector indexing
- Or start with keyword matching if your knowledge base is small

**4. Structured Output**
- JSON or YAML schema matching your summary requirements
- Include confidence scores and source references (which page/section)

## Minimal Working Stack

```python
# Rough pseudocode
pdf_text = extract_text(pdf_path)
chunks = split_by_section(pdf_text)

results = []
for chunk in chunks:
    extracted = extract_with_rules(chunk)  # Fast
    enriched = llm_extract(chunk)           # Slower, selective use
    matched = cross_reference_kb(extracted + enriched)
    results.append(matched)

summary = aggregate_results(results)
```

## Key Practical Decisions

| Choice | Reasoning |
|--------|-----------|
| **Extract → Search** not **Search → Extract** | Prevents hallucination, anchors findings to actual content |
| **Hybrid extraction** (rules + LLM) | Rules handle 70% fast; LLM handles edge cases |
| **Local embeddings** (if possible) | Avoids API costs at scale; tools like `sentence-transformers` are lightweight |
| **Structured schema early** | Define what "key information" means before building |

## Gotchas

- **PDF layouts vary wildly**: test on actual samples early. Some PDFs are scanned images—you'll need OCR
- **LLM cost creep**: use token limits and caching to control costs
- **Cross-reference false positives**: add a review step initially (even manual spot-checks prevent bad data propagating)

## Scaling Later

Once this works, you can add:
- Async processing (queue system like Celery)
- Vector DB (Pinecone, Weaviate) if KB grows
- Fine-tuned extraction models

**For now: keep it synchronous, single-machine, with clear handoff points between steps.** This makes debugging and validation much easier.

```python
from anthropic import Anthropic

class DocumentAnalyzer:
    def __init__(self, kb_client):
        self.client = Anthropic()
        self.kb = kb_client
        self.file_id = None
    
    def upload_document(self, pdf_path: str):
        """Upload PDF once via Files API"""
        with open(pdf_path, "rb") as f:
            response = self.client.beta.files.upload(
                file=(f.name, f, "application/pdf")
            )
        self.file_id = response.id
        return self.file_id
    
    def extract_chunk(self, page_range: str):
        """Extract structured data from page range"""
        response = self.client.messages.create(
            model="claude-opus-4-6",
            max_tokens=4096,
            messages=[{
                "role": "user",
                "content": [
                    {
                        "type": "document",
                        "source": {"type": "file", "file_id": self.file_id}
                    },
                    {
                        "type": "text",
                        "text": f"Extract key info from {page_range}..."
                    }
                ]
            }],
            output_config={
                "type": "json_schema",
                "json_schema": {
                    "name": "extraction",
                    "schema": extraction_schema,
                    "strict": True
                }
            }
        )
        return json.loads(response.content[0].text)
    
    def cross_reference(self, extracted_data: dict):
        """Look up extracted entities in knowledge base"""
        for entity in extracted_data.get("entities", []):
            matches = self.kb.search(entity["name"])
            entity["kb_matches"] = matches
        return extracted_data
    
    def summarize(self, all_extractions: list):
        """Generate final structured summary"""
        response = self.client.messages.create(
            model="claude-opus-4-6",
            max_tokens=2048,
            messages=[{
                "role": "user",
                "content": f"Summarize: {json.dumps(all_extractions)}"
            }],
            output_config={
                "type": "json_schema",
                "json_schema": {
                    "name": "summary",
                    "schema": summary_schema,
                    "strict": True
                }
            }
        )
        return json.loads(response.content[0].text)
    
    def process(self, pdf_path: str):
        """End-to-end pipeline"""
        self.upload_document(pdf_path)
        
        # Process in chunks
        extractions = []
        page_ranges = [
            "1-20", "21-40", "41-60", "61-80", "81-100"
        ]
        
        for page_range in page_ranges:
            extracted = self.extract_chunk(page_range)
            crossed_ref = self.cross_reference(extracted)
            extractions.append(crossed_ref)
        
        final_summary = self.summarize(extractions)
        return final_summary
```

```typescript
import { NextRequest, NextResponse } from "next/server";

const client = new Anthropic();

export async function POST(request: NextRequest) {
  const { message, context } = await request.json();

  const stream = await client.messages.create({
    model: "claude-3-5-sonnet-20241022",
    max_tokens: 1024,
    stream: true,
    messages: [

  // Convert to ReadableStream for streaming response
  const encoder = new TextEncoder();
  const readableStream = new ReadableStream({
    async start(controller) {
      for await (const event of stream) {
        if (
          event.type === "content_block_delta" &&
          event.delta.type === "text_delta"
        ) {
          controller.enqueue(encoder.encode(event.delta.text));
        }
      }
      controller.close();
    },
  });

  return new NextResponse(readableStream, {
    headers: {
      "Content-Type": "text/event-stream",
```

```typescript
  const [isLoading, setIsLoading] = useState(false);

    try {

      const decoder = new TextDecoder();

      while (true) {
        const { done, value } = await reader!.read();

        // Parse Server-Sent Events
        const chunk = decoder.decode(value);
        const lines = chunk.split("\n");

            }
          }
        }
      }

      setMessages((prev) => [...prev, { role: "assistant", content: assistantMessage }]);

    }
  }, [messages]);
```
