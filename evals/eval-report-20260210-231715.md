## Core Components

**1. Document Processing Layer**
```
PDF → Text Extraction → Chunking → Storage
```
- Use **PyPDF2** or **pdfplumber** for extraction (good balance of simplicity/reliability)
- Split into ~500-1000 token chunks with 100-token overlap to preserve context
- Store chunks in a simple SQLite database initially (scales up later if needed)

**2. Information Extraction**
```
Chunks → LLM → Structured JSON → Validation
```
- Use Claude API or similar with a clear prompt specifying output format
- Extract: key entities, dates, amounts, decisions, dependencies
- Batch process chunks (cheaper, less latency than individual calls)
- Include validation step to catch hallucinations

**3. Cross-Reference Module**
```
Extracted Data → Query Knowledge Base → Flags/Enrichment
```
- Start with a simple JSON or CSV knowledge base
- Query for matches on entities (company names, regulations, precedents)
- Flag inconsistencies or missing context
- Lightweight—don't over-engineer this yet

**4. Summary Generation**
```
Structured Data → LLM → Final Report
```
- Feed validated extractions + cross-reference results to a final summarization prompt
- Generate structured output (JSON or markdown)

```python
# Minimal viable tech stack
```

## Core Design


## Why This Approach

**Files API** (not re-uploading per request):
- Upload PDFs once, reference by ID across multiple requests
- 500MB per file limit (handles your 100-page PDFs easily)
- Reduces token waste and latency

**Claude for analysis** (Opus 4.6 or Sonnet 4.5):
- Single-pass extraction + reasoning in one call
- 200K context handles complex documents with knowledge base context
- Structured outputs guarantee JSON schema compliance

**Batch processing** (for cost):
- If you have volume, batch requests at 50% cost
- Process overnight or during off-peak hours

```python
    accuracy_v, tone_v, style_v = specialist_verdicts
```

```typescript
"use client";

import { useState } from "react";

export default function ChatBox() {
  const [input, setInput] = useState("");
  const [messages, setMessages] = useState<
    Array<{ role: "user" | "assistant"; content: string }>
  >([]);
  const [isLoading, setIsLoading] = useState(false);

  const handleSubmit = async (e: React.FormEvent) => {
    e.preventDefault();
    if (!input.trim()) return;

    const userMessage = input;
    setInput("");
    setMessages((prev) => [...prev, { role: "user", content: userMessage }]);
    setIsLoading(true);

    try {
      const response = await fetch("/api/chat", {
        method: "POST",
        headers: { "Content-Type": "application/json" },
        body: JSON.stringify({
          messages: [...messages, { role: "user", content: userMessage }],
        }),
      });

      if (!response.ok) throw new Error("API request failed");

      let assistantMessage = "";
      const reader = response.body?.getReader();
      const decoder = new TextDecoder();

      if (reader) {
        while (true) {
          const { done, value } = await reader.read();
          if (done) break;

          const chunk = decoder.decode(value);
          const lines = chunk.split("\n");

          for (const line of lines) {
            if (line.startsWith("data:")) {
              try {
                const data = JSON.parse(line.slice(5));
                if (
                  data.type === "content_block_delta" &&
                  data.delta?.type === "text_delta"
                ) {
                  assistantMessage += data.delta.text;
                  setMessages((prev) => {
                    const updated = [...prev];
                    if (
                      updated[updated.length - 1]?.role === "assistant"
                    ) {
                      updated[updated.length - 1].content = assistantMessage;
                    } else {
                      updated.push({
                        role: "assistant",
                        content: assistantMessage,
                      });
                    }
                    return updated;
                  });
                }
              } catch (e) {
                // Skip invalid JSON
              }
            }
          }
        }
      }
    } catch (error) {
      console.error("Chat error:", error);
    } finally {
      setIsLoading(false);
    }
  };

  return (
    <div className="w-full max-w-2xl mx-auto p-4">
      <div className="space-y-4 mb-4 h-96 overflow-y-auto border rounded p-4">
        {messages.map((msg, i) => (
          <div
            key={i}
            className={`p-3 rounded ${
              msg.role === "user"
                ? "bg-blue-100 ml-auto max-w-xs"
                : "bg-gray-100 mr-auto max-w-xs"
            }`}
          >
            {msg.content}
          </div>
        ))}
      </div>

      <form onSubmit={handleSubmit} className="flex gap-2">
        <input
          type="text"
          value={input}
          onChange={(e) => setInput(e.target.value)}
          disabled={isLoading}
          placeholder="Ask a question..."
          className="flex-1 border rounded px-4 py-2"
        />
        <button
          type="submit"
          disabled={isLoading}
          className="bg-blue-500 text-white rounded px-6 py-2 disabled:opacity-50"
        >
          {isLoading ? "..." : "Send"}
        </button>
      </form>
    </div>
  );
}
```

```typescript
  const [input, setInput] = useState("");
  const [loading, setLoading] = useState(false);

  const handleSubmit = async (e: React.FormEvent) => {

      method: "POST",
      body: JSON.stringify({ messages: newMessages }),

    const decoder = new TextDecoder();

    try {
      while (true) {
        const { done, value } = await reader.read();
        if (done) break;

        const chunk = decoder.decode(value);
        // Parse Server-Sent Events (SSE) format
        const lines = chunk.split("\n");
        for (const line of lines) {
          if (line.startsWith("data: ")) {
            try {
              const event = JSON.parse(line.slice(6));
              if (event.type === "content_block_delta" && event.delta?.text) {
                assistantMessage += event.delta.text;
              }
            } catch (e) {
              // Skip malformed JSON
            }
          }
        }
      }
    } finally {
      reader.releaseLock();
    }

    setMessages([...newMessages, { role: "assistant", content: assistantMessage }]);
    setLoading(false);
  };

  return (
    <div className="p-4 max-w-2xl mx-auto">
      <div className="space-y-4 mb-4">

            className={`p-3 rounded ${
              msg.role === "user" ? "bg-blue-100" : "bg-gray-100"
            }`}
          >
            {msg.content}
          </div>
        ))}
      </div>
      <form onSubmit={handleSubmit} className="flex gap-2">
        <input
          type="text"
          value={input}
          onChange={(e) => setInput(e.target.value)}
          placeholder="Ask about your data..."
          className="flex-1 p-2 border rounded"
          disabled={loading}
        />
        <button
          type="submit"
          disabled={loading}
          className="px-4 py-2 bg-blue-500 text-white rounded disabled:opacity-50"
```

```python
def sort_dicts(
    data: List[Dict[str, Any]],
    key: str,
    reverse: bool = False,
    default: Any = None,
    skip_missing: bool = False
) -> List[Dict[str, Any]]:
    """
    Sort a list of dictionaries by a specified key.
    
    Args:
        data: List of dictionaries to sort
        key: Dictionary key to sort by
        reverse: If True, sort in descending order (default: False)
        default: Default value for missing keys (default: None)
        skip_missing: If True, exclude dicts without the key (default: False)
    
    Returns:
        Sorted list of dictionaries
    
    Raises:
        ValueError: If key doesn't exist and skip_missing is False with no default
    
    Examples:
        >>> data = [{'name': 'Alice', 'age': 30}, {'name': 'Bob', 'age': 25}]
        >>> sort_dicts(data, 'age')
        [{'name': 'Bob', 'age': 25}, {'name': 'Alice', 'age': 30}]
    """
    if not data:
        return []
    
    # Check if all dicts have the key
    has_key = [key in item for item in data]
    
    if not all(has_key):
        if skip_missing:
            # Filter out dicts without the key
            data = [item for item in data if key in item]
        elif default is None:
            # Raise error if key not found and no default provided
            missing_indices = [i for i, has in enumerate(has_key) if not has]
            raise ValueError(
                f"Key '{key}' not found in items at indices: {missing_indices}. "
                f"Provide a default value or set skip_missing=True"
            )
    
    # Sort with proper handling of missing keys
    return sorted(
        data,
        key=lambda x: x.get(key, default),
        reverse=reverse
    )
```
